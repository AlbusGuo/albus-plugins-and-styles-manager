# Plugins & Styles Manager

Plugins & Styles Manager 是一款面向 [Obsidian](https://obsidian.md) 的社区插件，旨在为第三方插件与 CSS 代码片段提供集中化的管理界面。适用于插件数量较多、需要分类整理或批量操作的使用场景。

> 仅支持桌面端。最低要求 Obsidian v1.11.4。

---


## 核心功能概览

| 功能模块 | 说明 |
|---|---|
| 插件管理器 | 以模态窗口形式集中展示所有第三方插件，支持启用/禁用、卸载、打开设置、添加备注 |
| CSS 片段管理 | 在同一界面管理所有 CSS 代码片段，支持创建、重命名、删除、启用/禁用、添加描述 |
| 分组系统 | 为插件和 CSS 片段创建自定义分组，支持自定义分组颜色，侧边栏实时统计各分组数量 |
| 搜索与筛选 | 按名称、作者、描述进行关键词搜索；按启用状态筛选；支持特殊搜索语法 |
| 命令面板集成 | 为每个插件和 CSS 片段自动注册切换命令，可通过命令面板或快捷键快速启用/禁用 |
| 状态持久化 | 管理器窗口关闭后保留搜索词、筛选状态、分组选择和滚动位置，再次打开时恢复现场 |

## 安装

### 手动安装

1. 从 [Releases](https://github.com/AlbusGuo/albus-obsidianx/releases) 页面下载最新版本的 `main.js`、`manifest.json` 和 `styles.css` 三个文件。
2. 在 Vault 目录下创建文件夹：
   ```
   <Vault>/.obsidian/plugins/albus-psm/
   ```
3. 将上述三个文件放入该文件夹。
4. 重启 Obsidian（或在 **设置 --> 社区插件** 中点击"重新加载"）。
5. 在 **设置 --> 社区插件** 中找到 **Plugins & Styles Manager** 并启用。

## 使用指南

### 打开插件管理器

有两种方式打开管理器界面：

- 点击左侧功能区（Ribbon）的包裹图标。
- 打开命令面板（`Ctrl/Cmd + P`），搜索并执行 **"打开插件与样式管理器"**。

管理器以模态窗口呈现，左侧为分组侧边栏，右侧为内容列表。

---

### 插件管理

管理器右侧列表展示所有已安装的第三方插件，每个插件条目包含以下信息与操作：

**信息展示**

- 插件名称（仅桌面端可用的插件会显示"桌面端"标识）
- 分组标签（带自定义颜色）
- 版本号与作者
- 用户自定义备注

**可用操作**

| 操作 | 说明 |
|---|---|
| 启用/禁用开关 | 切换插件的启用状态 |
| 设置按钮 | 跳转至该插件的设置页面（仅在插件已注册设置页时显示） |
| 分组按钮 | 通过弹出菜单将插件分配到指定分组 |
| 卸载按钮 | 卸载插件（需二次确认） |
| 备注编辑 | 点击备注区域即可编辑，按 `Enter` 保存，支持 `Shift+Enter` 换行 |

---

### CSS 代码片段管理

在侧边栏切换到"CSS 代码片段"分组即可进入片段管理视图。

**可用操作**

| 操作 | 说明 |
|---|---|
| 启用/禁用开关 | 切换片段的加载状态 |
| 新建片段 | 工具栏右侧的"+"按钮，创建空白 CSS 文件并自动以系统默认编辑器打开 |
| 重命名 | 点击片段名称进入编辑模式，按 `Enter` 确认或 `Escape` 取消 |
| 打开文件 | 使用系统默认应用打开 `.css` 源文件 |
| 删除 | 从磁盘移除片段文件（需二次确认） |
| 描述编辑 | 点击描述区域编辑，操作方式与插件备注一致 |
| 分组按钮 | 将片段分配到指定的 CSS 分组 |

---

### 分组系统

分组系统用于将插件和 CSS 片段进行分类管理。插件分组与 CSS 片段分组各自独立。

**内置分组**

每类资源包含两个不可删除的内置分组：

- **全部** -- 显示所有条目（忽略分组过滤）
- **其他** -- 未分配到任何自定义分组的条目的默认归属

**自定义分组管理**

在 **设置 --> Plugins & Styles Manager** 中管理分组：

- **添加分组**：点击"添加插件分组"或"添加 CSS 分组"按钮。
- **重命名分组**：点击分组名称进入内联编辑模式，按 `Enter` 保存或 `Escape` 取消。
- **自定义颜色**：每个分组配有颜色选择器，所选颜色将作为管理器中分组标签的背景色。可随时通过重置按钮恢复默认颜色。
- **删除分组**：删除后，原属该分组的所有条目将自动移至"其他"分组。

**侧边栏实时统计**

管理器侧边栏的每个分组项旁会显示当前筛选条件下的匹配数量。搜索词、状态筛选的变化会实时反映在各分组的计数上。

---

### 搜索与筛选

**关键词搜索**

工具栏搜索框支持按以下字段进行模糊匹配：

- 插件：名称、作者、用户备注
- CSS 片段：名称、用户描述

**状态筛选**

搜索框右侧的状态按钮支持三种状态循环切换：

| 图标状态 | 含义 |
|---|---|
| 圆圈 | 显示全部 |
| 勾选圆圈 | 仅显示已启用 |
| 叉号圆圈 | 仅显示未启用 |

**特殊搜索语法**

在搜索框中输入 `???` 可查找所有未添加备注/描述的条目。该功能便于批量补充缺失的描述信息。

## 命令面板命令列表

以下命令均可通过 `Ctrl/Cmd + P` 调出命令面板后按名称搜索执行，也可在 **设置 --> 快捷键** 中绑定自定义快捷键。

| 命令名称 | 说明 |
|---|---|
| 打开插件与样式管理器 | 打开 Plugins & Styles Manager 窗口 |
| 切换插件: \<插件名\> | 为每个已安装的第三方插件自动注册，快速切换其启用/禁用状态 |
| 切换CSS片段: \<片段名\> | 为每个 CSS 代码片段自动注册，快速切换其加载状态 |

---

## 设置项说明

在 **设置 --> Plugins & Styles Manager** 中可配置以下内容：

**插件分组**

管理插件分组列表。支持添加、重命名、删除分组以及自定义分组标签颜色。

**CSS 片段分组**

管理 CSS 片段分组列表。功能与插件分组一致。
